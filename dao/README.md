# snglsDao

Use Node.js [8.10.0](https://itnext.io/nvm-the-easiest-way-to-switch-node-js-environments-on-your-machine-in-a-flash-17babb7d5f1b)
```sh
nvm use 8.10.0
```

## Install
```sh
npm install
```

## Deploy DAO
```sh
npm run start
```
